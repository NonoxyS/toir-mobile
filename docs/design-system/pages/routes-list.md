# Page Override: Список маршрутов

> Переопределяет MASTER.md для главного экрана «Назначенные маршруты».

## Специфика экрана

Первый экран после входа. Техник должен мгновенно видеть: есть ли незавершённые маршруты,
какой активен прямо сейчас, и сколько ждёт синхронизации.

## Карточка маршрута

```
┌─ elevation.1 ─────────────────────────────────────┐
│ ПТО-7 · Насосная станция №3     [В работе ▶]      │
│ Маршрут #1234                                      │
│ ████████░░░░  5 / 12 точек                         │
│ 📅 04 апр 2026  ⏱ ~40 мин                         │
└────────────────────────────────────────────────────┘
```

### Структура карточки

- Padding: 16dp (spacing.md) со всех сторон
- Radius: `radius.md` (10dp)
- Gap между карточками: 12dp

**Строка 1** (space-between):
- Название объекта: `type.headline` (17sp, 600), `color.textPrimary`
- Статус-бейдж: справа, компактный

**Строка 2**:
- Номер маршрута: `type.bodyMedium` (14sp), `color.textSecondary`

**Прогресс-бар**:
- Линейный, height 4dp, margin-top 10dp
- Background: `color.border`
- Fill: завершённые → `color.success`; текущие → `color.warning`
- Подпись: `type.caption` (12sp, 400), `color.textSecondary` — «X / Y точек»

**Строка метаданных** (иконка + текст):
- Дата: иконка `calendar` 16dp + date `type.caption`
- Разделитель · между мета-полями
- Время: иконка `clock` 16dp + duration `type.caption`
- Все: `color.textSecondary`

### Состояния карточки

| Состояние | Отличие |
|-----------|---------|
| `Не начат` | opacity 0.7, бейдж серый |
| `В работе` | полная яркость, бейдж warning ▶ |
| `Завершён` | фон-тинт `successSubtle` на карточке, бейдж success ✓ |
| `Ожидает синхр.` | левая полоска 3dp `color.sync`, иконка ⇅ |

## Порядок карточек

1. Маршруты «В работе» — сверху
2. Маршруты «Не начат» — по дате начала
3. Маршруты «Завершён, не синхр.» — ниже
4. Маршруты «Завершён» — последними

## Синхронизация в App Bar

Правый край App Bar:
- `color.success` чек → всё синхронизировано
- `color.warning` ⇅ → есть ожидающие
- `color.error` ✕ → ошибка синхронизации
- Анимированный spin при активной синхронизации

Тап на иконку → открывает экран «Статус синхронизации».

## Пустые состояния

**Нет маршрутов:**
```
      [иконка clipboard 48dp, textDisabled]
  Нет назначенных маршрутов
  Обновите список или обратитесь к диспетчеру
      [Обновить]  ← Secondary кнопка
```

**Ошибка загрузки:**
```
      [иконка cloud-off 48dp, error]
  Не удалось загрузить маршруты
      [Повторить]  ← Primary кнопка
```

**Offline, нет кэша:**
```
      [иконка wifi-off 48dp, textDisabled]
  Нет данных для офлайн-работы
  Подключитесь к сети для синхронизации
```
